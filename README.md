# PF2019
Pour Feliciter 2019 game inspired by Sokoban.

You can play the game at: https://georgik.rocks/html5-game-sokoban-pf2019/

## Technology

- Vue.js
- Vuex
- Vue Router

## Level design

Levels are described in `js/levels.js` in simple text form:

 ```
 "wwwwwww",
 "wa    w",
 "w ox  w",
 "wwwwwww",
 ```
 
- `a` - actor
- `o` - light bulb
- `x` - lightning spot
- `w` - wall
